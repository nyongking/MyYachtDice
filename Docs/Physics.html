PhysicsLib

외부 의존 없이 C++ 표준 라이브러리만으로 구현한 3D 리지드바디 물리 엔진. Sphere, Box, Plane 충돌체를 지원하며, SAT(Separating Axis Theorem) 기반 충돌 감지와 Sequential Impulse + Baumgarte Stabilization 기반 충돌 해결을 제공한다. Island 분리를 통한 병렬 솔빙과 IJobDispatcher 인터페이스를 통한 멀티스레드 확장이 가능하다.

C++17 Header-only Math Zero Dependencies Semi-implicit Euler SAT Collision Sequential Impulse Baumgarte Stabilization Island-based Parallel Solving Union-Find Trigger / Layer Mask

🏗️아키텍처 개요

PhysicsLib은 5개 계층으로 구성된다: 수학 타입(Vec3, Quat, Mat3) → 물리 프리미티브(RigidBody, Collider, Contact) → 충돌 감지(BroadPhase, NarrowPhase) → 충돌 해결(IslandBuilder, ContactSolver) → 월드 오케스트레이터(PhysicsWorld). 모든 수학 연산은 인라인 함수로 제공되며, 외부 라이브러리 의존이 전혀 없다.

IntegratePosition 구현 보기 
void RigidBody::IntegratePosition(float dt)
{
    if (bodyType != BodyType::Dynamic)
        return;

    // 속도 클램프 — 과도한 충돌 반응으로 인한 폭발 방지
    float linSpeedSq = Vec3LengthSq(linearVelocity);
    if (linSpeedSq > maxLinearSpeed * maxLinearSpeed)
        linearVelocity *= maxLinearSpeed / sqrtf(linSpeedSq);

    float angSpeedSq = Vec3LengthSq(angularVelocity);
    if (angSpeedSq > maxAngularSpeed * maxAngularSpeed)
        angularVelocity *= maxAngularSpeed / sqrtf(angSpeedSq);

    positionWorld += linearVelocity * dt;

    orientation = QuatNormalize(
        orientation + Quat(angularVelocity.x, angularVelocity.y,
                           angularVelocity.z, 0.f)
                    * orientation * 0.5f * dt);
}

💥충돌 감지

BroadPhase Class

AABB 기반 브루트포스 브로드페이즈. 모든 충돌체 쌍을 순회하며 3단계 필터링을 적용한다: (1) 레이어 마스크 호환성, (2) Plane이면 무조건 통과, (3) AABB 겹침 테스트. O(n²) 복잡도로 향후 공간 분할 자료구조로 교체 가능.

메서드

⭐ void GetPairsBruteForce(const vector<Collider*> colliders, vector<ColliderPair>& out)

내부 헬퍼 (private)

양방향 레이어 마스크 검사. (1 << a.layer) & b.mask AND (1 << b.layer) & a.mask.

3축 AABB 겹침 테스트. 각 축에서 min ≤ max 조건 확인.

NarrowPhase Class

BroadPhase가 걸러낸 ColliderPair에 대해 정밀 충돌 테스트를 수행한다. 타입 쌍에 따라 적절한 Collision 함수로 디스패치하며, isTrigger 플래그에 따라 contacts 또는 triggers 벡터로 분류한다.

메서드

⭐ void TestPair(const ColliderPair& pair, vector<Contact>& contacts, vector<Contact>& triggers)

충돌체 쌍의 타입을 정렬(Sphere < Box < Plane)한 뒤 매칭되는 Collision 함수를 호출. 결과 Contact에 colA/B, bodyA/B를 채워 넣는다.

타입 정렬 규칙: Sphere(0) < Box(1) < Plane(2) 순서로 정렬하여 항상 작은 타입이 a, 큰 타입이 b가 되도록 보장한다. 이를 통해 5가지 조합만 처리하면 된다.

ContactSolver Class Solver

⭐ void ResolveVelocity()

모든 constraint를 순회하며: (1) 법선 충격량 계산 + 누적 클램핑(≥0), (2) 접선 충격량 계산 + 쿨롱 마찰 클램핑(|λT| ≤ μ·λN). 각 충격량을 즉시 적용.

유효질량(Effective Mass) 공식: K = m_A⁻¹ + m_B⁻¹ + dot(cross(I_A⁻¹ · cross(rA, n), rA), n) + dot(cross(I_B⁻¹ · cross(rB, n), rB), n)
effectiveMass = 1 / K
Baumgarte Stabilization: bias = restitution_bias - (β/dt) · max(depth - slop, 0). slop은 미세 관통을 허용하여 지터(jitter)를 방지한다.

void ContactSolver::ResolveVelocity()
{
    for (auto& cc : m_constraints)
    {
        RigidBody* a = cc.ct->bodyA;
        RigidBody* b = cc.ct->bodyB;
        const Vec3& normal = cc.ct->normal;

        Vec3 vRel = b->GetVelocityAtPoint(cc.ct->point)
                   - a->GetVelocityAtPoint(cc.ct->point);

        // ── Normal impulse ──
        float vN = Vec3Dot(vRel, normal);
        float lambdaN = -(vN + cc.bias) * cc.effectiveMassNormal;

        float oldN = cc.accumulatedImpulseN;
        cc.accumulatedImpulseN = fmaxf(oldN + lambdaN, 0.f);  // clamp ≥ 0
        lambdaN = cc.accumulatedImpulseN - oldN;

        Vec3 impulseN = normal * lambdaN;
        a->ApplyImpulseAtPoint(-impulseN, cc.ct->point);
        b->ApplyImpulseAtPoint( impulseN, cc.ct->point);

        // ── Tangent impulse (friction) ──
        vRel = b->GetVelocityAtPoint(cc.ct->point)
             - a->GetVelocityAtPoint(cc.ct->point);

        float vT = Vec3Dot(vRel, cc.tangent);
        float lambdaT = -vT * cc.effectiveMassTangent;

        float limit = cc.mixedFriction * cc.accumulatedImpulseN;
        float oldT = cc.accumulatedImpulseT;
        cc.accumulatedImpulseT = clamp(oldT + lambdaT, -limit, limit);
        lambdaT = cc.accumulatedImpulseT - oldT;

        Vec3 impulseT = cc.tangent * lambdaT;
        a->ApplyImpulseAtPoint(-impulseT, cc.ct->point);
        b->ApplyImpulseAtPoint( impulseT, cc.ct->point);
    }
}

void PhysicsWorld::Step(float dt)
{
    auto* dp = m_dispatcher ? m_dispatcher : &m_defaultDispatcher;

    // Step 1: 중력 적용 + 속도 적분 (위치는 아직 갱신하지 않음)
    dp->ParallelFor(0, (int)m_bodies.size(), [&](int i) {
        RigidBody* body = m_bodies[i];
        if (body->IsDynamic())
            body->ApplyForce(m_gravity * (1.f / body->GetInvMass()));
        body->IntegrateVelocity(dt);
        body->UpdateWorldInertia();
    });

    m_pairs.clear();
    m_collisions.clear();
    m_triggers.clear();

    // Step 2: 충돌 감지 (BroadPhase + NarrowPhase)
    m_broadPhase.GetPairsBruteForce(m_colliders, m_pairs);
    for (auto& pair : m_pairs)
        m_narrowPhase.TestPair(pair, m_collisions, m_triggers);

    // Step 3: Island 분리 + 병렬 접촉 해결
    if (!m_collisions.empty()) {
        m_islandBuilder.Build(bodyTemps, m_collisions);
        const auto& islands = m_islandBuilder.GetIslands();

        dp->ParallelFor(0, (int)islands.size(), [&](int i) {
            ContactSolver solver;
            solver.Resolve(islandContacts, dt);
        });
    }

    // Step 4: 솔버가 보정한 속도로 위치/회전 갱신
    dp->ParallelFor(0, (int)m_bodies.size(), [&](int i) {
        m_bodies[i]->IntegratePosition(dt);
    });

    // Step 5: 어큐뮬레이터 클리어
    dp->ParallelFor(0, (int)m_bodies.size(), [&](int i) {
        m_bodies[i]->ClearAccumulators();
    });
}
